# Clipper Digital Prototypes

Static, hosted prototypes for design and stakeholder review.

| Prototype | Description | URL |
|---|---|---|
| [`blue-print/`](./blue-print/) | BluePrint app — instant print-ad mockup builder. Light + dark mode. Click-through wizard with mocked backend. | [Open](https://clipper-digital.github.io/prototypes/blue-print/) |

## Adding a new prototype

1. Create a new folder at the repo root (e.g., `my-prototype/`)
2. Drop a self-contained static site inside (HTML / CSS / JS / assets)
3. Add an entry to the table above
4. Commit, push, and the new prototype is live at `https://clipper-digital.github.io/prototypes/my-prototype/`
